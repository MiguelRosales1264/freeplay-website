Block vs Inline Tags

Block tags are tags that take up the full width of the page. They are used to create large sections of content. It's like a box

Inline tags are tags that only take up the width of the content inside of them. They are used to create small sections of content. They just flow with the rest of the content.

Div and span are neutral container elements in HTML. Esentailly they're use is to contain other tags and seperate them from other text for later use in CSS. 'div' is block-level tag. 'span' is inline-level tag.
