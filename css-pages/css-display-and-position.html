Fixed vs Fluid Widths - The Foundations of Website Layout

Height and Overflow

When you set a height value on a container, the content inside will not adjust to the height value. Instead, it will overflow out of the container.

Here the content overflows the container. To resolve this we use the 'overflow' property

The overflow has four values: visible, hidden, scroll, and auto.
